5.24. 職業構造体解析

5.24.1. 解析
5.24.1.1. $C4179E 職業構造体
5.24.2. TODO リスト

職業構造体（a.k.a. レベルアップ構造体）$C4179E のメンバー解析を行う。

5.24.1. 解析

職業構造体は、言うまでもなくレベルアップ時の処理に密接な関係を持つ。

5.24.1.1. $C4179E 職業構造体

アドレス $C4179E からすべての職業構造体データが定義されている。 ひとつの構造体データが占めるデータサイズは #$0173 byte と非常に巨大だ。 これが 9 個配列されている。 注意しなければならないのは、職業構造体にはいわゆるヌルデータの概念がないという事実だ。 ID がゼロの職業は、普通に有効なデータ（戦士のもの）である。

また、アドレス $C424A9 からは、各データの先頭アドレスを格納する配列となっている。 ただし、$C424A9 (1byte) だけはデータバンク #$C2 を意味する。

職業構造体のメモリレイアウトは以下のようになっている。 巨大な構造体であるので、記述を一部簡略化してある。

文字列 ID

職業の名前を表す文字列 ID である。

グラフィック 男

この職業の男キャラクターのキャラクターグラフィック構造体 ($C893D4) の ID である。

以下の同類フィールドの説明ではいちいち断らないが、 サブルーチン $C63084 を解析すればわかるように、 この値はそのキャラクターが死んでいるとき、 ぬいぐるみ着用時、 レムオル系使用時や、 各種イベント時（例えばロマリア王様モード）等は使用されない。

グラフィック 女

上述「グラフィック 男」フィールドの女キャラクター版である。

グラフィック 水着

この職業の女キャラクターが、以下のどれかを装備している場合に表示されるキャラクターグラフィックの、 キャラクターグラフィック構造体 ($C893D4) の ID である。

  • あぶないみずぎ
  • しんぴのビキニ
  • まほうのビキニ
未解析

未解析。

各職業に対して、非ゼロの値が格納されていることから、 未使用ではない可能性が強い。 遺憾ながら、記者はこのフィールドを参照しているコードを発見することができなかった。

初期性格決定テーブルの ID [0..4]

この職業のキャラクターを仲間登録所で新規に作成する際に参照される。 性格条件群の決定の項以下を参照。

習得呪文配列先頭アドレス

この職業のキャラクターが、レベルアップ時に習得する可能性のあるレベルと、 習得呪文のペアを定義する配列のアドレスである。 厳密には、呪文 ID を直接保持するわけではなく、 $7E3925 構造体データの一部である「習得済み呪文フラグ」の位置を指示する配列 ($C456AC) のインデックスを保持する。

フラグと呪文の対応関係を解析するのであれば、 まずはサブルーチン $C458C8 から始めて、 呼び出しの深いほうに解析していくとよい。

習得呪文配列サイズ

上述「習得呪文配列」のサイズである。 例えば賢者の習得呪文配列サイズは、魔法使いのそれと僧侶のそれの和に等しい。 しかし、三者の習得呪文配列はいずれも異なる。

レベル N 経験値

この職業のキャラクターが、レベル N であるための必要最低限の経験値である。 N は 2 から 99 までの値である。

男キャラ Lv 1 のちから

この職業の男キャラクターを仲間登録所で新規に作成する際、ちからパラメータの初期値となる。 つまり、タネを使ってそのキャラクターのパラメータを上げる前に、 ウィンドウで確認できる値の 16 倍の値である。

女キャラ Lv 1 のちから

上述「男キャラ Lv 1 のちから」フィールドの女版である。

ちから成長 レベル境界値[0..4], ちから成長幅 [0..4]

レベルアップ時に再設定されるちからパラメータの値（の基準値）を計算するのに用いられるフィールドである。 例えば、この職業のキャラクターのレベルアップ直後のパラメータ基準値は、 キャラクターのレベルが「レベル境界値 [k] を越えていて、かつレベル境界値 [k + 1] 未満」と仮定の下で、 以下のように評価される。 

基準値 = (上述レベル 1 の値
        + 成長幅[0]     * (レベル境界値[0]     - 2)
       + 成長幅[1]     * (レベル境界値[1]     - レベル境界値[0])
       + ...  
       + 成長幅[k - 1] * (レベル境界値[k - 1] - レベル境界値[k - 2])
       + 成長幅[k]     * (現在レベル           - レベル境界値[k]))
       / 16
