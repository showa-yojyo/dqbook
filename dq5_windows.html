3.8. ウィンドウ

3.8.1. ウィンドウ定義表

本節ではウィンドウシステムについて考察したい。 残念ながら現時点では解析不足につき、読者にウィンドウ仕様の詳細を理解していただくだけの十分な議論を展開できないでいる。 当面の間は記者が独自に調査したウィンドウ ID とその内容の対応表を掲載するに留めざるを得ない。

3.8.1. ウィンドウ定義表

次の各アドレスから始まるウィンドウに関するデータ列をひとつの表にまとめた。 ただし、この表は本書の他の節で採るような解析手法とは異なる怪しい手段によって調査した結果から得たものなので、品質を向上する余地がある。 表の最初の列はウィンドウ ID であり、真ん中の二列は各列名に対応するアドレスに実際に格納されている値であり、 最後の列は記者が目視で確認したウィンドウの内容を粗く表現した記述だ。 このような粗雑なデータであっても、別の逆アセンブリコード解析の助けになる。

  • $04EB48: ウィンドウデータ格納アドレス配列
  • $04EC70: 何らかの配列
  • $04ED04: ウィンドウデータ
